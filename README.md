# Modul151
Repo für die B-Kompetenzen im Modul 151

#### Installation:
XAMPP herunterladen und installieren

1. Lade dir die aktuelle Version von XAMPP für Windows von der offiziellen [Apache Friends Webseite](https://www.apachefriends.org/) herunter.
2. Starte den Installer.
3. Folge dem Assistenten und achte darauf, dass **Apache** und **MySQL** ausgewählt sind.

Projekt in XAMPP integrieren

Um das Projekt über den Apache-Server laufen zu lassen, muss es im `htdocs`-Ordner liegen:

1. Navigiere zu `C:\xampp\htdocs\`.
2. Erstelle dort einen neuen Ordner für dieses Projekt (z. B. `Modul151-projekt-name`) oder clone das Repository direkt dorthin:
   ```bash
   git clone [https://github.com/DEIN-BENUTZERNAME/DEIN-REPOSITORY.git](https://github.com/DEIN-BENUTZERNAME/DEIN-REPOSITORY.git)

Projekt in Visual Studio Code öffnen

1. Navigiere im Windows-Explorer zu `C:\xampp\htdocs\`.
2. Mache einen **Rechtsklick** auf den Projektordner.
3. Wähle **"Mit Visual Studio Code öffnen"**
